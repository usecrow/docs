## Verify connection

After installation, verify the MCP server is connected:

1. Open Cursor Settings (`Cmd/Ctrl + ,`)
2. Navigate to **Cursor Settings â†’ Tools & MCP**
3. Confirm "crow" appears in the list of connected servers

To test, try asking Cursor: *"How do I embed the Crow widget? Use MCP tools."*

## What you can do

With Crow MCP connected, you can ask Cursor to:

- Help integrate the Crow widget into your website
- Set up identity verification for authenticated users
- Configure OpenAPI or MCP server integrations
- Understand how to use Crow's features
